OpenCode for Copilot Chat 使用你自己的 OpenCode Go API Key，让 GLM-5.2、Kimi K2.7、DeepSeek V4、MiMo、MiniMax、Qwen 出现在模型选择器中。

只需粘贴一次，之后可通过命令面板更新或移除。

- `Cmd/Ctrl + Shift + P`：打开命令面板
- `OpenCode: 设置 API Key`：设置或更新 API Key
- `OpenCode: 清除 API Key`：移除 API Key
- `OpenCode: 获取 API Key`：打开 opencode.ai/auth 获取 API Key
- `OpenCode: 查询用量`：打开 OpenCode Go 用量控制台
