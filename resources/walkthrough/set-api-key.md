OpenCode for Copilot Chat uses your own OpenCode Go API key to make GLM-5.2, Kimi K2.7, DeepSeek V4, MiMo, MiniMax, and Qwen available in the model picker.

Paste it once, then update or remove it later from the Command Palette.

- `Cmd/Ctrl + Shift + P`: Open the Command Palette
- `OpenCode: Set API Key`: Set or update your API key
- `OpenCode: Clear API Key`: Remove your API key
- `OpenCode: Get API Key`: Open opencode.ai/auth to get your API key
- `OpenCode: Query Usage`: Open the OpenCode Go usage console
